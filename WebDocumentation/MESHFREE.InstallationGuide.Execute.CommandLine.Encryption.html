Encryption

Encrypts files to share UCVs and CVs MESHFREE can work with but cannot be read by a human

To encrypt files please check the CommandLine section If MESHFREE cannot find the given UCVs and/or CVs MESHFREE will automatically search for the give name with the appendix '.enc' For example if no specific CV and UCV name was given the two files MESHFREE looks for are common_variables.dat and USER_common_variables.dat If one of those is not found MESHFREE looks for common_variables.dat.enc and USER_common_variable.dat.enc If those encrypted files are not found either MESHFREE will exit with an error message. It is currently not possible to \include_Ucv{} an encrypted file into an encrypted file. It is possible to include multiple encrypted files into an unecrypted file via \include_Ucv{} Note: the name of the file need to be without the .enc, generally you should never specify the .enc ending into any of your parameters as those will be automatically found once there is no file found with the original name.
