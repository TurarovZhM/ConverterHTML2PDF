rotate{

rotate the given geometry about a point with a rotation axis and angle

include{File} ... rotate{O_x, O_y, O_z, Phi_x, Phi_y, Phi_z, OPTIONAL:RotationAngle } ...
The geometry is rotated about the point (O_x, O_y, O_z) with the rotation vector (Phi_x, Phi_y, Phi_z). The vector (Phi_x,Phi_y,Phi_z) provides the rotation axis. If RotationAngle is NOT given, then the length of the vector (Phi_x,Phi_y,Phi_z) provides the angle of rotation in radians. If RotationAngle is given, then the length of (Phi_x,Phi_y,Phi_z) does not play any role. MESHFREE will (internally) normalize this vector and apply the rotation angle given in the optional variable RotationAngle. Warning: If the length of the vector (Phi_x,Phi_y,Phi_z) is zero, no rotation can be effected.
