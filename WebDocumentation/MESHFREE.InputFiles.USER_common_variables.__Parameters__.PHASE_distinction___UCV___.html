PHASE_distinction

invoke detection of interface connections (UCV)

PHASE_distinction = 'YES'
Default: PHASE_distinction = 'NON' Setting this parameter to 'YES', invokes detection of interphase connections. Each boundary point (also free surface point) searches for another boundary point of a different chamber, which is close enough and with which it can exchange interphase boundary conditions, see BCON_CNTCT. If a contact point is found, the index of this point is stored in %ind_iopp%.
This item is referenced in:
%ind_iopp% iopp=index of opposite point.
