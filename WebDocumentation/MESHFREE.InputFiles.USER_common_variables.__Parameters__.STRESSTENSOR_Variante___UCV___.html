STRESSTENSOR_Variante

version of stress tensor time integration (UCV)

STRESSTENSOR_Variante = 7
Default: STRESSTENSOR_Variante = 3 TODO
option description
7 Tiny elastic steps and reprojection onto yield curve after each tiny step
6 dS/dt = mue*(D+D^T)/2 + R*S - S*R using mueElasticGuess
5 dS/dt = mue*(D+D^T)/2 + R*S - S*R
4, 41, 42, 3, 31, and 32 Provided the fact that N = exp(2*S/mue), the differential equation for integration of N is dN/dt = N*D^T + D*N which can be integrated analytically. Then, after integration, N can be transformed back by S = (mue/2)*log(N)
2 ???
any other integer Same as 3.
