SimCut

(chamberwise) parameter to stop filling of geometry by MESHFREE points after a certain number of filling cycles

SimCut = 10
Default: SimCut = 0 If a positve number is given, MESHFREE will perform the given number of filling iterations for interior points. Then it generates an output as defined in Saving and Results, after that MESHFREE stops. Note: This parameter can also be set chamberwise for multiphase simulations (see also KindOfProblem, CHAMBER). If it is not set for specific chambers, it is automatically set according to the non-chamberwise definition for all chambers. Special feature:
SimCut = -10
With a negative number for SimCut, MESHFREE will perform abs(SimCut) filling cycles for interior points. After this, the simulation is started regularly.
