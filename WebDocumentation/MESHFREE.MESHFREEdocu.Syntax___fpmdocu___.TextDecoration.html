TextDecoration

give colors of emphasis on the text

"#red This text is in red red#" while this is in normal color "#blue This text is in blue blue#" while this is in default color "#green This text is in green green#" while this is in default color "#orange This text is in orange orange#" while this is in default color "#em This text is in italic em#" while this is in default style "#bold This text is bold bold#" while this is in default style "#! This text is also bold !#" while this is in default style "#UCVCODE" and "UCVCODE#" mark beginning and end of a USER_common_variable-example, that will be rendered by coloring: - $Variables$ : dark green - &Alias& : dark read - #comments : yellow here an example:
# some comment in this line BC_v($DedicatedBND$) = ( %BND_slip% , &AliasForFriction&, &AliasForMomentumControl& ) # comments maybe also here INITDATA($Material$, %ind_p_dyn%) = [ equn($EQNinitp$) ] # and here
"#mono" and "mono#" mark the beginning and end of a monospace block. All blanks and returns are interpreted each by each. Long lines are not cut, but there will be a horizontal scrollbar if needed. For example.
1. My first item 1.1. The first subitem is indented by 3 blanks since I typed three blanks 1.1.1 The subsubitem is indented by 8 blanks This long line is maybe too long to bring on the screen therefore a horizontal scrollbar will automtically appear. If you do not like th acrollbar, wrap the line by hand and maybe wrap again for shorter lines and even shorter. 1.2 2. plapla
"#numberedlist" and "numberedlist#" represent beginning and end of numbered list "#bulletlist" and "bulletlist#" represent beginning and end of bullet list "#item" represents individual item of either numbered list or bullet list Example:
"#numberedlist #item first item #numberedlist #item first.first item numberedlist# #item second item #numberedlist #item second.first item #numberedlist #item second.first.first item numberedlist# numberedlist# numberedlist#" will display like
  2. first item
    2. first.first item
  3. second item
    2. second.first item
      2. second.first.first item
"#bulletlist #item first bullet #item second bullet bulletlist#" will display like
  • first bullet
  • second bullet
