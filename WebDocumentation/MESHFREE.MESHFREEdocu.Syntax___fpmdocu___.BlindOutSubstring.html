BlindOutSubstring

blinds out a marked substring such that it is not rendered in the html-document

Use the icon "___" in order to mark beginning and end of the string to be blinded out. Example: for instance, the boundary conditions for velocity and pressure use many key-words in common. BC_v(...) = ( %BND_wall%, ... ) BC_p(...) = ( %BND_wall%, ... ) In order to distinguish between the two in the MESHFREEdocu, on could set up
'!$FPMDOCu #h MESHFREE.InputFiles.USER_common_variables.BoundaryConditions.BC_v.%BND_wall%___BC_v___ h#' '!$FPMDOCu #h MESHFREE.InputFiles.USER_common_variables.BoundaryConditions.BC_p.%BND_wall%___BC_p___ h#'
The true short names are still "%BND_wall%___BC_v___" and "%BND_wall%___BC_p___", such that MESHFREEdocu can distingiuish between them, however they will both be displayed as "%BND_wall%"
