Table-block

write the lines into a table

The icon "#Table" opens and the icon "Table#" closes a Table block. Everything that is in between will be written into columns and rows in a Table by the html-output. The columns are seperated by a '|' symbol and each new line is considered as a row. The Table can be created in the documentation as follows, Example 1: head_column 1|head_column 2|head_column 3 column 4|column 5|column 6 column 7|column 8|column 9

Example 2: heading col 1 |heading col 2 this is test 1|this is test 2 this is test 4|this is test 5
