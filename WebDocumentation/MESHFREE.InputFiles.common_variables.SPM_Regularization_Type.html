SPM_Regularization_Type

regularization type if all boundaries are Neumann-type

The PDEs solved in MESHFREE for the pressure are of the form with the boundary conditions (Dirichlet) or (Neumann). If, for the whole domain, only Neumann-conditions are given, the arising linear sparse system is singular and has to be regularized.
