%BND_none%

treat the boundary point as if it would be an interior point

ATTENTION: this type of boundary condition is, theoretically, not valid in general if used with FLIQUID_ConsistentPressure_Version =?1?? , i.e. a 1 at the second digit. Here, the accelarations are considered to be quasistationary, hence %BND_none% will make a mistake if used in a non-quasistationary setting.
