Unpublished bugfixes for StableCandidate_R2018.1.(n+1)
R2019.1.0

This version is the same as beta19.10.1. For a detailed list of differences compared to R2018.1 (based on beta18.05.0), see Beta.

New:
  • improvements to GASDYN solver.
  • Improvements to DROPLETPHASE solver.
  • Redlich-Kwong Gas Law for modelling real gas behaviour (in LIQUID solver, see PhysicalProperties).
  • New feature %HEAT_EQ_1D_BC% for using 1D heat equation as temperature boundary condition (see HeatEquation1D).
  • New feature RepresentativeMassAlgorithm.
  • Improved MPI Shared Memory support, e.g. for GEOTREE2 and MONITORPOINTS (see COMP_SharedMemoryForBE, COMP_SharedMemoryForGT2).
  • Improved support for ERFHDF5.
  • Support for version 4.1 .msh files for triangular 2D meshes from Gmsh.
  • Performance improvements for geometry operations, see GeometryOperations.
  • new manipulate{ - feature: extrude{
  • TWOSIDED is now deprecated, use duplicate{ and thickenabs{ instead
  • New option for ReadInPointCloud: writeOut, see WriteOutManipulations
  • New INTEGRATION feature Skip.
  • New EVENT handles %EVENT_WriteResume% and %EVENT_SaveResults%
  • New CommandLine option --exec-dir, -e.
  • New CommandLine option --clparam, -clp to be used as system variable @CLPARAM@.
Bugfixes:
  • Multiple bugfixes and improvements for stability and performance.
R2018.1.1
Bugfixes:
  • Fix bug where INTEGRATION values were all zero when organization steps were skipped.
  • Clarify behavior for CHAMBER. If no CHAMBER in alias is given, then CHAMBER=MAT.
  • Fix compilation on HLRS Cray.
  • Pass SPM_N_Iterations to SAMG.
R2018.1.0
This version is based on beta18.05.0 including important bug fixes up to beta18.06.0, see Beta New:
  • MPI shared memory for boundary elements, see COMP_SharedMemoryForBE
  • MEMORIZE functionality (Note: does not work with restart)
  • Improvements to ERFHDF5
  • STANDBY-pointclouds, see EnSight .
  • New features in INTEGRATION: SelectionFeatures, SelectByPercentileBounds, %ASSIGN_FUNCTIONVALUE%, SequentialFiltering
  • New features for BE_MONITOR_ITEM, SAVE_BE_MONITOR_ITEM: %CUMU_ASSIGN%, %CUMU_SMOOTH%, %CUMU_SMOOTH_AreaBased%
  • New features for geometry: removeBEonCondition{, coarsenGeometry{, support for quads in OBJ reader (see include{)
  • New feature for EVENT: %EVENT_FunctionManipulation%
  • New checkpoint/resume feature (see checkpoint )
  • Fallback for LaunchRestart
  • Improvements for Variables: @SystemVariable@, alias-in-alias &MMM(&NNN&)&
  • ContinuationLines
  • New parameter SAVE_PrecisionTimestepFile
  • New parameter RepeatCurrentTimeStep_AdditionalComputationsAfterDataTransfer
  • New Equations feature: modulo operation mod()
  • CODI-execution order defined by appearance in the USER_common_variables.dat
  • step-by-step-execution
  • New common variables: TIMECHECK_Level, IGES_HealCorruptFaces, ISOLATEDPOINTS_ClusterOnResultingVolume, BE_CleanUp_STL, ORGANIZE_FuzzyMPIFilling, GEOTREE2_BND_FinalBoxSize, GEOTREE2_BND_FinalBoxDimension, delaunay_reduction, iFPM_process_ID (see also MESHFREE.ProgramExecution.Linux.Run.CommandLine), SAVE_ASCII_split
  • New options for FLIQUID_ConsistentPressure_Version
  • changed default values for GEOTREE2_FinalBoxSize: 4->16, GEOTREE2_MaximumBoxSize: 8->20
Bugfixes:
  • Multiple bugfixes and improvements for stability and performance.
